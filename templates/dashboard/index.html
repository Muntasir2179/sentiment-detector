Introducing Emotion
Classifier Model

I have trained a several Machine Leading Model on a kaggel dataset.
The models we trained are Naive Bayes, Support Vector Classifier(svn), Random Forest Classifier, Logistic Regression, K-nearest neighbours(KNN) and Decision Tree Regression.
The Logistic Regression model has shown better accuracy. The trained model can now predict emotion from a text sequence.
There are six type of emotion that the model is able to predict

Apart from all the machine learning models, I have also fine tuned BERT (Bidirectional Encoder Representations from Transformers) on the same dataset. It has outperformed all the machine learning models with achieving accuracy of 95%.

Considering the computational power, I have choosen the Logistic Regression model over BERT, although Logistic Regression achieved 93% accuracy which is less that BERT. I made such choice because, machine learning model is more efficient choice when computational power is concerned.
